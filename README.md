
  # Restaurant Reservation Web App

  This is a code bundle for Restaurant Reservation Web App. The original project is available at https://www.figma.com/design/2g1w0ExRZsxEfJqmPrRxOG/Restaurant-Reservation-Web-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  If you use Supabase: copy `.env.example` to `.env` and fill in `VITE_SUPABASE_PROJECT_ID` and `VITE_SUPABASE_ANON_KEY` (from Supabase → Settings → API).

  Run `npm run dev` to start the development server.
